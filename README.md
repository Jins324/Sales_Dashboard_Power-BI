# Sales Performance Dashboard

## Project Overview
This Power BI dashboard analyzes sales performance using KPIs, charts, and interactive slicers.

## Dashboard Features
- Total Sales
- Total Orders
- Total Customers
- Average Sales
- Sales by Region
- Monthly Sales Trend
- Top Products
- Sales by Category
- Interactive Slicers

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Dataset
- 200 Sales Records
- Customer Data
- Product Data
- Region Data
